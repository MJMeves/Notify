To run website:
1) Install node.js
2) download website folder
3) go into terminal and navigate to the website folder
4) run npm install
5) Take the sql file in the website folder and run that in your MySql to create the db and tables
6) Edit the server.js file under the comment that says "SQL Connection" and change the password field to your root password for mySql
7) In the terminal, run node server.js
